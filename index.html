sc-loop

closed-loop AI sound design with SuperCollider

[ goal / target sound ]
|
▼
LLM writes synths/*.scd
|
▼
render.sh → renders/*.wav (offline NRT)
|
▼
analyze.py → pitch, brightness, envelope…
compare.py → diff vs target
|
▼
LLM reads numbers, edits code, repeats

What it is

  • An LLM writes SuperCollider synth code
  • Renders it offline — no speakers needed
  • "Hears" the result through an audio analyzer
  • Iterates until the sound matches a target

Quick start

./scripts/render.sh synths/kick.scd renders/kick.wav 1
python3 tools/analyze.py renders/kick.wav
python3 tools/compare.py renders/kick.wav targets/ref.wav

Then open Claude Code in this folder and say e.g.
"make a dark metallic drone, iterate until centroid < 900 Hz"

Setup

  • SuperCollider.app in /Applications
  • pip3 install -r requirements.txt
